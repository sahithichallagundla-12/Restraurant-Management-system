System Requirements
===================

1. C++ Compiler
   - GCC (g++) 7.0 or higher
   - OR Clang/LLVM
   - OR Microsoft Visual C++ (MSVC)

2. Operating System
   - Windows 10/11
   - Linux (any distribution)
   - macOS

3. Build Tools
   - Make (optional, but recommended)
   - CMake (optional)

4. Development Environment
   - Terminal/Command Prompt/PowerShell
   - Text Editor or IDE (VS Code, Visual Studio, Code::Blocks, etc.)

Installation Instructions
=========================

Windows:
--------
1. Download and install MinGW (includes g++):
   - Visit: https://www.mingw-w64.org/
   - Or use: https://www.msys2.org/

2. Add g++ to PATH:
   - Set environment variable or use full path

3. Verify installation:
   g++ --version

Linux (Ubuntu/Debian):
---------------------
sudo apt-get update
sudo apt-get install build-essential g++
g++ --version

macOS:
------
1. Install Xcode Command Line Tools:
   xcode-select --install

2. Verify:
   g++ --version

Project Dependencies
